안녕하세요, 코리아포트원 개발팀입니다.

항상 저희 코리아포트원 서비스를 이용해주셔서 감사드리며,
**포트원 V1 REST API 응답 관련 변경 사항**이 있어 아래와 같이 안내드립니다.

<Highlight text="1.변경 사항" />

- V1 결제내역 단건조회 API에 신규 파라미터가 추가됩니다.

  변경 이후 **테스트 채널을 통한 결제건**은 기본적으로 **응답에서 제외** 됩니다

<Highlight text="2.대상 API" />

- V1 결제내역 단건조회 API

  (결제내역 단건조회 API를 제외한 다른 API에는 적용되지 않습니다.)

<Highlight text="3.신규 파라미터 사양" />

| 이름                | 타입        | 기본값     | 상세 설명                                                       |
| ----------------- | --------- | ------- | ----------------------------------------------------------- |
| `include_sandbox` | `boolean` | `false` | 테스트 채널 결제(`sandbox=true`)를 조회 결과에 포함할지 여부를 지정합니다.           |
|                   |           |         | **`false`**: 테스트 결제는 조회되지 않으며, 해당 건에 대해선 **404** 응답이 반환됩니다. |
|                   |           |         | **`true`**: 테스트 결제를 포함하여 조회합니다(테스트 환경 사용 시 권장).             |

<Highlight text="4. 동작 변화 요약" />

| 호출 상황        | 변경 전  | 변경 후          |
| ------------ | ----- | ------------- |
| 실운영 채널 결제 조회 | 정상 조회 | 변경 없음 (정상 조회) |
| 테스트 채널 결제 조회 | 정상 조회 | 404 응답        |

<Highlight text="5. 적용 일정" />

- **2026년 1월 26일 월요일**

<Callout icon="💡" title="참고사항" content="이번 변경은 운영중인 서비스에서 테스트 채널을 악용한 결제 변조(어뷰징)을 방지하기 위한 조치의 일환으로 테스트 채널을 기존 서비스 운영에 이용하지 않고 있다면 조치가 필요하지 않습니다.

단, 테스트 환경에서는 신규 파라미터를 이용하여 조회가 가능하도록 조치가 필요합니다." />
