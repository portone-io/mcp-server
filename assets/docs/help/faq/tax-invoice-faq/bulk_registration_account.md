거래처를 하나씩 등록하는 대신, CSV 또는 엑셀 파일을 업로드하여 대량으로 일괄 등록할 수 있는 기능입니다. \[거래처 관리] 페이지에서 일괄 등록 양식을 다운로드 받은 뒤, 데이터를 기입하여 업로드하면 자동으로 콘솔에 반영됩니다.

1. \[거래처 관리] 페이지에서 \[거래처 일괄 등록] 버튼을 눌러주세요.
2. \[일괄 등록 양식 다운로드] 버튼을 눌러 양식 파일을 다운로드해 주세요.
3. 다운로드한 파일을 열고 거래처 정보를 기입해 주세요. 필수 정보와 선택 정보를 구분하여 정확하게 입력합니다.
4. 파일을 저장한 뒤, \[파일 첨부] 버튼을 눌러 양식을 업로드해 주세요.
5. 업로드 시 사업자 정보 조회 및 휴폐업 상태 조회(미조회, 영업중, 휴업중, 폐업) 실행 여부를 선택할 수 있습니다.
6. \[등록] 버튼을 눌러 거래처 등록을 완료해 주세요.
