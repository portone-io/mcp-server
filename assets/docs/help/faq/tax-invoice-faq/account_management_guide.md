거래처 관리는 세금계산서 정발행 및 역발행 시 활용할 수 있는 거래처의 사업자 정보를 등록하고 관리하는 기능입니다. \[세금계산서] 내 \[거래처 관리] 메뉴에서 거래처를 등록하면, 세금계산서 작성 시 거래처 검색을 통해 공급받는자 정보를 간편하게 설정할 수 있습니다.

거래처 등록 시 아래 정보를 입력합니다.

| 구분     | 항목                     | 필수 여부      |
| ------ | ---------------------- | ---------- |
| 기본 정보  | 아이디                    | 자동 생성      |
|        | 사업자등록번호                | 필수         |
|        | 상호                     | 선택         |
|        | 대표자명                   | 필수         |
|        | 업태                     | 선택         |
|        | 업종                     | 선택         |
|        | 사업장 주소                 | 선택         |
| 대표 담당자 | 이름                     | 필수         |
|        | 전화번호                   | 선택         |
|        | 이메일                    | 필수         |
|        | 메모                     | 선택         |
| 추가 담당자 | 이름, 전화번호, 이메일, 메모      | 선택 (최대 5명) |
| 기타     | 국세청 연동 상태              | 자동관리       |
|        | 영업 상태 (미조회/영업중/휴업중/폐업) | 자동 관리      |
|        | 거래처 메모                 | 선택         |

거래처 등록은 개별 혹은 일괄로 등록할 수 있습니다. 개별 등록은 \[거래처 관리] 에서 거래처 정보를 직접 입력하여 등록할 수 있습니다. 일괄 등록은 CSV 또는 엑셀 파일을 업로드하여 거래처를 대량으로 등록합니다. 등록 시 페이지 하단에서 사업자등록번호 기준 사업자 정보 조회 및 휴폐업 상태 조회 실행 여부를 선택할 수 있습니다.

### 거래처를 등록하면 어떤 기능을 이용할 수 있나요?

- 세금계산서 작성 시 거래처 검색 : 거래처를 검색하여 공급받는자 정보를 자동으로 설정할 수 있습니다.
- 추가 담당자 자동 반영 : 거래처에서 설정한 추가 담당자 정보가 세금계산서 작성 시 자동으로 반영됩니다.
- 국세청 연동 : 역발행 세금계산서 이용을 위해 거래처 단위로 국세청 연동을 진행할 수 있습니다.
