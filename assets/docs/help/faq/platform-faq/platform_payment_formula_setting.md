지급금액은 정산정책, 수수료, 세금(면세액, 부가세, 원천징수세) 등을 모두 고려하여, 셀러에게 최종으로 지급되는 금액입니다. 지급금액 계산식은 아래와 같습니다.

#### 지급금액 = 지급예상금액 - 원천징수세액

- 지급예상금액 = 과세 공급가액 + 면세금액 + 부가가치세(VAT)
- 원천징수세액 = 소득세(3%) + 지방소득세(0.3%) \* 원천징수 대상자만 해당
