주문 취소로 인해 정산이 취소될 때 ‘주문 취소 정산’이라는 별도의 데이터를 생성하여 정산 상세 조회 페이지에서 취소 이력을 확인할 수 있습니다.

‘주문 취소 정산’은 정산 상세 조회에서 확인할 수 있습니다.

주문 취소 정산 API를 호출할 때 주문 내역을 기준으로 정산 취소 금액을 자동으로 계산하여 정산 취소 내역을 만듭니다.
